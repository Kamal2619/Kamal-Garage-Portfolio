PROJECT 05

420 Clothing Streetwear

Project Type Apparel Design
Duration 3 Weeks
Focus Area Streetwear / Graphic Design
Deliverables T-Shirt Graphics / Print Assets / Production-Ready Artwork

The Challenge

420 Clothing envisioned a streetwear collection that embraced controlled chaosgraphics with enough character to command attention while remaining versatile for everyday wear. The objective was to establish a signature visual identity that balanced bold artistic expression with refined composition, allowing each garment to feel distinctive without overwhelming the wearer.

The Solution

We developed a collection of original apparel graphics built around the philosophy of restraint within boldness. By combining symbolic imagery, surreal compositions, and intentional spacing, each design achieved visual impact while preserving wearability. Every graphic underwent multiple iterations to refine its balance, resulting in a cohesive collection that strengthened 420 Clothing's brand identity and positioned its apparel as conversation starting pieces rather than trend-driven prints.

Client Feedback
