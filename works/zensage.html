PROJECT 03

ZENSAGE

Project Type Apparel Graphics
Duration 5 Weeks
Focus Area Illustration / Anime Streetwear
Deliverables Hand-Drawn Graphics / Print-Ready Assets / Apparel Design Collection

The Challenge

Zensage wanted an original anime-inspired apparel collection that celebrated the culture without relying on overused visuals. The challenge was to create hand crafted illustrations that felt authentic to anime fans while giving the brand a distinct visual identity. Every design needed to evolve through rapid collaboration, balancing originality, storytelling, and premium execution.

The Solution

Every graphic was illustrated from scratch, beginning as hand-drawn sketches before being refined through continuous feedback with the Zensage team. By combining recognizable anime influences with original compositions, expressive line work, and a consistent visual language, we built a collection that felt both familiar and unmistakably unique. The collaborative workflow allowed ideas to evolve quickly, resulting in artwork that strengthened the brand's identity while creating designs made to spark conversation long before launch.

Client Feedback
